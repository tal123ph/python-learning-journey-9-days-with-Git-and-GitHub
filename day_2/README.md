# Day 2: Conditional Statements

This folder contains 6 Python programs demonstrating different ways to use conditional logic.

## List of Programs

1.  **`01_check_positive.py`**
    *   **Concept**: Simple `if` statement.
    *   **Description**: Checks if a given number is strictly greater than zero.

2.  **`02_odd_even.py`**
    *   **Concept**: `if-else` statement.
    *   **Description**: Determines whether a number is odd or even using the modulo operator (`%`).

3.  **`03_grade_system.py`**
    *   **Concept**: `if-elif-else` ladder.
    *   **Description**: Categorizes a numerical score into letter grades (A, B, C, D, F).

4.  **`04_nested_conditions.py`**
    *   **Concept**: Nested `if` statements.
    *   **Description**: Demonstrates decision making where a condition (having a license) is checked only if a previous condition (age) is met.

5.  **`05_ternary_operator.py`**
    *   **Concept**: Ternary operator (Conditional Expressions).
    *   **Description**: Shows how to write a concise if-else check in a single line of code (e.g., `value if true else value`).

6.  **`06_logical_operators.py`**
    *   **Concept**: Logical Operators (`and`, `or`, `not`).
    *   **Description**: Combines multiple conditions to make complex decisions, such as determining ticket discounts.
